# SmartHome_Manager
A C++ based project for sem 2 :)

In a modern household, managing multiple electronic utilities(lights,AC) and critical safety sensors (smoke alarms,motion detectors) manually is inefficient and prone to human error. There is a need for a centralized,automated system that can monitor energy consumption,handle safety alerts,and provide a unified interface for the user.

Our application addresses :

1. CENTRALIZED CONTROL : Instead of using separate apps or manual switches, our system provides a unifies interface to manage both energy consuming utilities and safety critical sensors.
2. ENERGY OPTIMIZATION : It tracks pattern usage allowing users to identify high consumption areas and reduce utility bills.
3. SCALABILITY: This system is designed so that a user can add 10 or 100 new devices without rewriting the core management logic.


This project demonstrates the power of Object Orient Programming in C++.
By integrating systems into a single framework, we have created a model for efficient home management.
